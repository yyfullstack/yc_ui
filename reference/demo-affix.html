YC Design 固钉组件

固钉组件是一种可以将元素固定在页面特定位置的 UI 组件。当页面滚动时，固钉元素会保持在固定位置，为用户提供持续的导航或操作入口。

主要应用场景包括：

• 固定导航栏 - 使导航始终可见

• 侧边工具栏 - 提供常用操作入口

• 目录导航 - 方便在长页面中跳转

• 返回顶部 - 快速回到页面顶部

组件特性：

• 支持多种固定位置（上下左右四角）

• 可设置偏移量

• 支持状态变化回调

• 平滑的显示/隐藏动画

• 支持暗色主题

YC Design 还提供了丰富的 UI 组件，帮助开发者快速构建高质量的用户界面。每个组件都经过精心设计，支持主题定制，并提供一致的使用体验。
