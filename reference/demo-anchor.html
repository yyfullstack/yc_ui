第一章：组件库简介

YC Design 是一套企业级 UI 设计语言，提供了丰富的组件库，帮助开发者快速构建高质量的 Web 应用。

组件库遵循一致性原则，确保所有组件在外观和行为上保持统一，为用户提供一致的使用体验。

本组件库包含按钮、表单、布局、导航、数据展示等多个类别的组件，满足各种业务场景需求。

第二章：设计原则

设计原则是组件库的基础，指导着每个组件的设计与实现。

一致性：组件在相似场景下表现一致，用户无需重新学习。

可复用性：组件设计为通用形式，可在不同项目中复用。

可访问性：组件支持键盘导航和屏幕阅读器，方便残障人士使用。

第三章：按钮组件

按钮是界面中最常用的交互元素之一，用于触发操作或提交表单。

按钮组件支持多种类型：主要按钮、次要按钮、文本按钮、链接按钮等。

同时支持不同的尺寸和状态：正常、悬停、激活、禁用等。

第四章：表单组件

表单组件用于用户输入和提交数据，是应用中不可或缺的组成部分。

包括输入框、选择器、开关、日期选择器等多种类型。

每个表单组件都支持验证、禁用、只读等状态，并提供清晰的错误提示。

第五章：布局组件

布局组件用于组织页面的整体结构，提供响应式的页面排版方案。

包括容器、栅格、间距、对齐等工具类。

支持移动端、平板端、桌面端等多种屏幕尺寸的适配。
