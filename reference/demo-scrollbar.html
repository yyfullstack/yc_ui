YC Design 滚动条组件

这是一个基础滚动条组件的演示。滚动条会在鼠标悬停时显示，方便用户滚动查看超出容器的内容。

滚动条组件提供了平滑的滚动体验，支持自定义样式和尺寸。组件遵循 YC Design 设计规范，与整体风格保持一致。

组件支持以下特性：

• 垂直滚动和水平滚动

• 多种尺寸变体（小、中、大）

• 自定义滚动条颜色

• 悬停和拖拽状态

• 平滑的过渡动画

• 主题适配（亮色/暗色）

滚动条组件适用于各种需要滚动内容的场景，如列表、表格、文本区域等。

原生滚动条

使用 CSS 实现原生滚动条样式

原生滚动条样式

通过 CSS::-webkit-scrollbar 可以自定义滚动条的样式，无需 JavaScript 库。

这种方式的优点是：

• 零依赖，纯 CSS 实现

• 性能更好

• 可以精确控制滚动条的每个部分

支持的伪元素：

• ::-webkit-scrollbar — 整个滚动条

• ::-webkit-scrollbar-track — 滚动条轨道

• ::-webkit-scrollbar-thumb — 滚动条滑块

• ::-webkit-scrollbar-button — 滚动条按钮（上下箭头）

• ::-webkit-scrollbar-corner — 滚动条角落

兼容性：Chrome、Safari、Edge 等 Chromium 内核浏览器。

滚动条各部分说明
webkit-scrollbar 整个滚动条容器
webkit-scrollbar-track 滚动条轨道（背景）
webkit-scrollbar-thumb 滚动条滑块（可拖动部分）
webkit-scrollbar-button 滚动条两端的按钮

暗色主题滚动条

在暗色背景下，滚动条使用半透明白色，更加协调。

滑块颜色从 30% 透明度过渡到 50% 透明度。

提供了良好的视觉对比度。

自定义颜色滚动条

主题色滚动条

自定义主题色滚动条

使用项目的主题色作为滚动条颜色。

轨道颜色使用主色的 20% 透明度。

滑块颜色使用主色。

悬停时使用主色的悬停色。

宽度设置为 10px，更加明显。

自定义颜色 CSS 变量
--yc-scrollbar-width 滚动条宽度（默认 6px）
--yc-scrollbar-thumb-width 滑块宽度
--yc-scrollbar-color 滚动条轨道颜色
--yc-scrollbar-thumb-color 滚动条滑块颜色
--yc-scrollbar-thumb-hover-color 滑块悬停颜色

滚动条状态

滚动条交互状态
默认状态 滚动条半透明或隐藏
悬停状态 滚动条滑块颜色变深
拖拽状态 按住滑块拖拽滚动内容
